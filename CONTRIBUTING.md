# Contributing to Chambers

Thank you for your interest in contributing to Chambers! We welcome contributions from the community to help improve the game logic and framework.

## How to Contribute

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your local machine.
3.  **Create a new branch** for your feature or bug fix:
    ```bash
    git checkout -b feature/amazing-new-card
    ```
4.  **Make your changes** and ensure everything is working.
5.  **Run Tests** to make sure you haven't broken existing logic:
    ```bash
    npm test
    ```
6.  **Commit your changes** with a clear message:
    ```bash
    git commit -m "Add: New Fireball card logic"
    ```
7.  **Push to your branch** and submit a **Pull Request (PR)**.

## Reporting Bugs

If you find a bug, please open an issue describing:
*   What happened.
*   What you expected to happen.
*   Steps to reproduce the error.
